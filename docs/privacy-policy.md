# Política de Privacidad de CV Generator
**Última actualización:** 19 de marzo de 2026
## Resumen
CV Generator es una extensión de navegador diseñada para ayudarte a crear currículums vitae profesionales. **Tu privacidad es nuestra prioridad.** Esta extensión:
- ✅ **NO recopila datos personales**
- ✅ **NO envía información a servidores externos**
- ✅ **NO utiliza cookies de seguimiento**
- ✅ **NO requiere registro ni cuenta de usuario**
- ✅ **Funciona completamente OFFLINE**
## 1. Información que procesamos
### 1.1 Datos que tú introduces
Cuando usas CV Generator, introduces información personal en el archivo `datos.yaml` (nombre, cédula, experiencia, etc.). **Esta información permanece exclusivamente en tu dispositivo local.** Nosotros no tenemos acceso a ella, no la almacenamos en servidores, y no la compartimos con terceros.
### 1.2 Archivos generados
Los archivos PDF y LaTeX que generas se crean localmente en tu computadora. Tú eres el único responsable de dónde los guardas, compartes o almacenas.
### 1.3 Almacenamiento local
La extensión utiliza el almacenamiento local de tu navegador para:
- Guardar tus archivos YAML recientes (opcional, tú decides si guardarlos)
- Recordar tus preferencias de configuración (tema, idioma, etc.)
Este almacenamiento es local y no se sincroniza con ningún servicio externo a menos que tú lo autorices explícitamente (por ejemplo, usando la sincronización de Edge con tu cuenta Microsoft).
## 2. Características específicas por navegador
### 2.1 Microsoft Edge
Si utilizas CV Generator en Microsoft Edge y tienes una sesión iniciada con tu cuenta Microsoft:
- **Edge Collections:** Si decides guardar tus plantillas en Collections, estos datos se sincronizarán según la configuración de privacidad de Microsoft.
- **Edge Drop:** Al compartir archivos mediante Drop, estos se transfieren a través de los servidores de Microsoft bajo su política de privacidad.
**Recomendación:** Revisa la [política de privacidad de Microsoft](https://privacy.microsoft.com/es-es/privacystatement) para entender cómo manejan tus datos en estos servicios.
### 2.2 Firefox
En Firefox, toda la información permanece estrictamente local. No se utiliza ningún servicio de sincronización a menos que tú actives manualmente la sincronización de Firefox.
### 2.3 Chrome
En Chrome, toda la información es local. No se utiliza la sincronización de Chrome para los datos de CV Generator.
## 3. Bibliotecas de terceros
CV Generator utiliza las siguientes bibliotecas de código abierto, todas cargadas localmente o desde CDNs seguros:
- **CodeMirror** - Editor de código (licencia MIT)
- **js-yaml** - Parseador YAML (licencia MIT)
- **Pyodide** - Python en WebAssembly (licencia Apache 2.0/Mozilla)
- **Font Awesome** - Iconos (licencia gratuita)
Estas bibliotecas pueden cargarse desde CDNs (jsdelivr.net, cdnjs.cloudflare.com) que tienen sus propias políticas de privacidad, pero **no se envía información personal** en estas solicitudes.
## 4. Seguridad
- **Código abierto:** Todo el código de CV Generator es público y verificable en [GitHub](https://github.com/donshuapps-cloud/cv-generator-monorepo).
- **Sin conexiones ocultas:** Puedes auditar el tráfico de red usando las herramientas de desarrollo de tu navegador.
- **Actualizaciones:** Las actualizaciones se distribuyen únicamente a través de las tiendas oficiales de extensiones (Chrome Web Store, Firefox Add-ons, Microsoft Store).
## 5. Tus derechos
Dado que no almacenamos tus datos, no es necesario ejercer derechos de acceso, rectificación o cancelación sobre nuestros sistemas. Tú controlas tus datos localmente.
Si tienes preguntas sobre esta política, puedes contactarnos en: **donshu.apps@gmail.com**
## 6. Cambios en esta política
Podemos actualizar esta política de privacidad ocasionalmente. Te notificaremos cualquier cambio publicando la nueva política en esta misma URL y actualizando la fecha de "Última actualización".
